

User:
    id -> int
    Имя Фамилия -> str
    login -> str
    password -> str(хешированный)
    Почта -> str
    аватар -> file //не обязательное поле 
    Роль пользователя (учитель/ученик/user) -> str
    Почта подтверждена -> Boolean
    Учитель (если роль учителя) -> one_to_one
    Ученик (если роль ученика) -> one_to_one
    Пользователь (если роль user) -> one_to_one

Пользователь:
    id -> int
    Этапы пройденные -> one_to_many relation //У одного пользователя несколько пройденных этапов
    Этапы не пройденные -> one_to_many relation //У одного пользователя несколько не пройденных этапов
    Сертификат -> bool (Доступен/Не доступен)
    user -> ForeignKey


Учитель:
    id -> int
    Классы -> one_to_many relation //у одного учителя может быть несколько классов
    Кастомные тесты созданные учителем -> one_to_many relation
    user -> ForeignKey

Ученик:
    id -> int
    login -> str 
    password -> str //не зашифрованный для отображения учителю
    результаты прохождения тестов -> one_to_many
    user -> ForeignKey




Класс:
    id -> int
    Учитель -> one_to_one // у одного класса может быть только один учитель
    Ученики -> one_to_many // У одного класса может быть несколько учеников
    Открытые классу тесты -> one_to_many // у одного класса может быть несколько открытых тестов

Этап 
    id -> int
    Название этапа -> str
    Краткое описание -> str
    картинка/иконка -> file
    Порядковый номер этапа -> int
    Информация? -> (пока не известно в каком виде она будет у нас храниться) str или file
    Тест -> ForeignKey relation
    Доступ без авторизации -> bool (Доступен/Не доступен)

Задание
    id -> int
    Тип вопроса -> str (будет выбираться из выпадающего списка)
    Текст вопроса -> str
    Кастомный -> boolean
    user -> ForeignKey
    Правильные ответы -> one_to_many relation
    Возможные ответы -> many_to_many relation

Тесты
    id -> int
    Название теста -> str
    Задания -> one_to_many 
    Кастомный -> boolean
    user -> ForeignKey
    

Ответы
    id -> int
    Текст ответа -> str
    Кастомный -> boolean
    user -> ForeignKey


Результат прохождения теста:
    id -> int
    Ученик -> one_to_one
    Тест -> one_to_one
    Ответы ученика -> one_to_many
    кол-во правильно отвеченных вопросов -> int
    Общее число вопросов -> int
    Процент правильных ответов к общему числу ответов -> int
